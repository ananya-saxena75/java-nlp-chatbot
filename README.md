# 🤖 Java NLP Chatbot

A simple NLP-based chatbot built using Java and Stanford NLP concepts.

## 🚀 Features
- Intent classification
- Basic NLP processing
- Dynamic responses
- JSON-based intents
- Console-based interaction

## 🛠 Tech Stack
- Java
- Maven
- NLP concepts

## 🎥 Demo (How it works)

Run locally:

```bash
mvn spring-boot:run
```

Then open:
http://localhost:8080

Example:
You: hello
Bot: Hello! How can I help you?
