# 🤖 Java NLP Chatbot

A simple NLP-based chatbot built using Java and Stanford NLP concepts.

## 🚀 Features
- Intent classification
- Basic NLP processing
- Dynamic responses
- JSON-based intents
- Console-based interaction

## 🛠 Tech Stack
- Java
- Maven
- NLP concepts

## ▶️ How to Run
```bash
mvn clean install
mvn exec:java -Dexec.mainClass="chatbot.ChatBotApp"
